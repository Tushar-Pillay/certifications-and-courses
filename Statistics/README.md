## 📘 Course Description
This course provides a strong foundation in **statistical thinking and data-driven
decision-making**, designed specifically for learners preparing for MBA programs.
It focuses on understanding data, probability concepts, and drawing reliable
inferences from limited samples.

The course emphasizes how statistical methods support business decisions
when complete population data is unavailable.
