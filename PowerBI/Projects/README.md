## 📘 Project Description
This guided project focuses on **data preparation and transformation using Power BI**.
It demonstrates how datasets from multiple sources can be imported, cleaned,
combined, and structured to support analytics and dashboard development.

The project reflects real-world Business Intelligence workflows where raw data
must be shaped, validated, and integrated before reporting and analysis.
