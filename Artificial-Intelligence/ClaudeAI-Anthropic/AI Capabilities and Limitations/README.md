This certification provides a deep technical mental model of how Large Language Models (LLMs) behave, moving beyond surface-level prompting to understand the underlying machine properties. It is the technical companion to the **4D Framework** (Delegation, Description, Discernment, Diligence).

#### 🧠 Core Learning: The 4 Machine Properties
I mastered the four fundamental properties that define the "capability-to-limitation" continuum of AI:

1.  **Next Token Prediction:** 
    *   Learned how models generate responses probabilistically.
    *   Understood why AI excels at reformatting and summarizing but can struggle with precise logic or niche data retrieval.
2.  **Knowledge Boundaries:** 
    *   Explored the difference between "internalized" training data and external retrieval.
    *   Studied why models produce "confidently incorrect" answers and how to identify high-risk hallucination zones.
3.  **Working Memory (Context Window):** 
    *   Mastered context management and the "lost in the middle" phenomenon.
    *   Learned to prioritize information so critical data doesn't "fall off the edge" of the model's active memory.
4.  **Steerability & Instructions:** 
    *   Developed techniques to precisely control model behavior through structural constraints (like XML tags).
    *   Explored the limits of instructional control when user intent clashes with model training.

#### 🛠️ Technical Competencies
*   **System Diagnostics:** Learned to analyze unexpected outputs, identify which machine property is failing, and apply targeted fixes instead of generic retries.
*   **Property Collisions:** Analyzed complex scenarios where multiple limitations overlap (e.g., long documents pushing working memory while requiring niche knowledge).
*   **Model Characterization:** Understood how fine-tuning and RLHF (Reinforcement Learning from Human Feedback) shape the "personality" and safety guardrails of Claude.
