# Certifications & Courses

This repository contains certificates from courses completed across
**Data Analytics, Product Analytics, SQL, Excel, Python, and related domains**.

The purpose of this repository is to showcase **continuous learning,
skill development, and verified credentials** in a structured and
easy-to-navigate format.

---

## 📂 Certificate Categories

The certificates in this repository are organized into the following sections.
Each section contains individual folders for every certificate.

- 📗 **Excel**
- 📊 **Power BI**
- 📈 **Statistics**
- 🤖 **Artificial Intelligence** 
- 🧮 **SQL**
- 🐍 **Python**
- 📊 **Business & Operations Analytics**
